-->RESTful Web API to demonstrate GET, PUT, POST, PATCH request on a ToDo item

The project is created using Java 8 on Eclipse Neon IDE(Windows 10) and tested on PostMan.

To import the project:
1.) Download/Clone the project using GitHub link and unzip.
2.)In Eclipse, import the project using new Maven project.

To run the project:
- Run the Application.java file as a Spring Boot App and you can test the test cases using PostMan.
